# Auto_battler

Python game project for a single player auto battler (with roguelite elements)

## Requirements
* Python 3.9 or higher
* pygame 2.3.0
* numpy 1.24.2
* pandas 2.0.0

### Optional Dependencies
* matplotlib 3.7.1
* scikit_learn 1.2.2

## Installation
```
git clone https://github.com/Junkkah/auto_battler.git
cd auto_battler
pip install .
python main.py
```
